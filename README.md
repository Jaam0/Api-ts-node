# Api Rest Node js, typescript and mysql

## Steps to run this api rest:

1. **npm install**
2. **Go to database directory from this project and run the script.sql**
3. **node  dist/app.js**

# Techonologies used in this project

1. **NodeJs**
2. **TypeScript**
3. **MySql**
